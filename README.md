## 📌 About The Project

**Mini Bank Management System** is a beginner-friendly banking project built using **Python**.

The project was first created as a **command-line interface application**, where the core banking logic was written manually. Later, the project was upgraded into a simple **Streamlit web application** with better validation, improved structure, and a cleaner user interface.

This project is made for learning and practice purposes.

---

## ✨ Features

* 🧾 Create a new bank account
* 🔐 Generate a unique account number
* 💰 Deposit money
* 🏧 Withdraw money
* 📄 Check account details
* ✏️ Update account details
* 🗑️ Delete account
* 📁 Store data using JSON file handling
* ✅ Validate PIN, age, amount, and balance
* 🌐 Simple Streamlit web interface

## ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/kamranakbar25/Mini-bank-management-system.git
```

Go inside the project folder:

```bash
cd Mini-bank-management-system
```

Install Streamlit:

```bash
py -m pip install streamlit
```

---

## ▶️ How To Run

Run the Streamlit app:

```bash
py -m streamlit run bank_management_system.py
```

If your file name is `app.py`, then run:

```bash
py -m streamlit run app.py
```

## 🔐 Important Note

This is a beginner-level learning project. It is **not made for real banking use**.

For safety:

* Do not upload real user data
* Do not upload `data.json`
* Do not store real PINs or passwords
* Use this project only for learning and practice
