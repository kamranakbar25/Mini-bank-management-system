# Mini Bank Management System 🏦

A beginner-friendly **Bank Management System** built with **Python** and upgraded into a simple **Streamlit web application**.

This project started as a command-line Python program where the core banking logic was written first. Later, AI assistance was used to polish the code, improve validations, and convert it into a Streamlit-based web app.

---

## 🚀 Features

* Create a new bank account
* Generate a unique account number
* Deposit money
* Withdraw money
* Check account details
* Update account details
* Delete account
* Store data using JSON file handling
* Simple Streamlit web interface
* Input validation for PIN, age, amount, and balance

---

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON
* Object-Oriented Programming

---

## ⚙️ Installation

```bash
git clone https://github.com/kamranakbar25/Mini-bank-management-system.git
cd Mini-bank-management-system
py -m pip install streamlit
```

---

## ▶️ How to Run

```bash
py -m streamlit run bank_management_system.py
```

If your main file name is `app.py`, then use:

```bash
py -m streamlit run app.py
```

## 🔐 Important Note

This is a beginner-level learning project. It is not made for real banking use.

For safety reasons:

* Do not upload real user data
* Do not upload `data.json`
* Do not store real PINs or passwords
* Use this project only for learning and practice
